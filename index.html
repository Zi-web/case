КАК МЫ НАУЧИМ ВАШ САЙТ ПРОДАВАТЬ ЛУЧШЕ?

В стоимость работ по улучшению юзабилити сайта входит:

  • Анализ информации о компании, выявление преимуществ, специфики компании; базовый (обзорный) анализ сайта с целью выявления наличия проблем и особенностей функционала.
  • Анализ технических особенностей сайта с точки зрения возможности внесения правок и исправления проблем на сайте.
  • Анализ сайтов конкурентов: выявление сильных и слабых сторон сайта Клиента по сравнению с его конкурентами.
  • Анализ и сегментирование целевой аудитории сайта (необходимо для создания адаптированного для нее функционала и вида сайта).
  • Разработка портретов пользователей, создание персонажей.
  • Разработка пользовательских сценариев для персонажей.
  • Анализ поведенческих показателей статистики сайта с целью выявления его наиболее проблемных мест (веб-аналитика).
  • Проверка сайта на соответствие ожиданиям пользователей и позиционированию компании (образу бренда).
  • Прохождение пользовательских сценариев и фиксация проблем.
  • Проверка кроссбраузерности и кроссплатформенности сайта.
  • Создание гипотез для проведения юзабилити-тестирования.
  • Проведение юзабилити-тестирования.
  • Оценка и анализ результатов юзабилити-тестирования и подготовка отчета.
  • Выявление эффективности сайта методом Eye-tracking.
  • Оценка результатов, полученных методом Eye-tracking, и подготовка отчета.
  • Проверка корректной работы функционала сайта, оценка его целесообразности на сайте. Выявление необходимости в добавлении функционала на сайт.
  • Экспертный анализ юзабилити сайта. Подготовка отчета по результатам экспертной оценки.
  • Обобщение результатов анализа сайта. Составление общих рекомендаций по улучшению юзабилити на сайте (и решению выявленных проблем).
  • Создание прототипов/макетов измененных (в соответствии с рекомендациями) страниц.
  • Настройка цели «Вовлеченность посетителей Сайта».
  • Настройка цели «Конверсия посетителей в потенциальных покупателей».
  • Согласование рекомендаций и прототипов (макетов) с клиентом.
  • Одобрение правок и утверждение макетов для внесения правок.
